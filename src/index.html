VSCode Features

Look here to see the features being demonstrated!

You can open up a folder to work on a project, using
File>Open Folder or Ctrl-O

The activity pane on the far left of the window opens up many useful views

  • File explorer
  • Search
  • Source Control (git)
  • Run & Debug
  • Extensions

The file explorer encourages working in multiple files

It's easy to create new files and folders

Install the official Python extension

Hovering over a function name shows the parameters and docstring. Ctrl-Click takes you to the function definition.

It shows you variables that you haven't used, and variables that aren't defined.

It will show you syntax errors and tell you what the problem is on hover.

It can autocomplete functions from modules and methods on objects.

f-strings are a way to easily insert variables into a string 


            h = 1.2
            f"She is {h}m tall"

Working with multiple files is easy with tabs and split windows

Right clicking on tabs gives many options for how to manage them

VSCode can help you importing from your own files

Ctrl-Click and autocomplete still work well
