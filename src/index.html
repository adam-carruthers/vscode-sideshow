VSCode Features

Look here to see the features being demonstrated!

The first thing that shows up when you open VSCode is a tutorial

It's really good, don't skip it!

You can open up a folder to work on a project, using
File>Open Folder or Ctrl-O

The activity pane on the far left of the window opens up many useful views

  • File explorer
  • Search
  • Source Control (git)
  • Run & Debug
  • Extensions

The file explorer encourages working in multiple files

It's easy to create new files and folders

Install the official Python extension

Hovering over a function name shows the parameters and docstring. Ctrl-Click takes you to the function definition.

It shows you variables that you haven't used, and variables that aren't defined.

It will show you syntax errors and tell you what the problem is on hover.

It can autocomplete functions from modules and methods on objects.

f-strings are a way to easily insert variables into a string 


            h = 1.2
            f"She is {h}m tall"

Working with multiple files is easy with tabs and split windows

Right clicking on tabs gives many options for how to manage them

VSCode can help you importing from your own files

Ctrl-Click and autocomplete still work well

Using F2 you can smart rename things

We can select our python environment with Ctrl-Shift-P or at the bottom right.

VSCode then uses that to help us run stuff in the right environment, and give us warnings about packages that aren't installed

Let's talk about git

Git clearly highlights the files which have been modified

And greys out files in the .gitignore

You can easily manage the branch you're on from the menu in the bottom left

The source control tab makes staging changes and committing easy

The GitLens extension adds lots of extra useful features like seeing who wrote which code

The GitHistory extension allows you to easily explore the timeline of changes

There's a three way merge editor, to easily fix merge conflicts

VSCode has a debugger, that allows you to walk through your code and see what's going on.

Even with DataFrames!

VSCode encourages testing, and there is an entire tab that makes it easy to run and explore your tests.

Pytest is the most popular Python library for doing testing, we just assert what we want to be true.

Phew...

VSCode can open Jupyter Notebooks

If you go down to the bottom pane, you can see the local variables (as in spyder)

VSCode can run python files like notebooks, with
# %%
lines marking cell boundaries

This avoids leaking data through the notebook.

The Edit CSV extension allows easy opening of CSV files

Markdown is the file type we use in README files, and VSCode understands it well

As does GitLab, some markdown even works in Microsoft Teams!

The DrawIO extension allows us to make flow chart diagrams to document process

Then we can easily use them directly in our Markdown files!!

Type annotations document data types of variables, parameters and returns 


              names: list[str] = [
                "Amaan", "Mia"
              ]
              def add(
                a: float, b: float
              ) -> float:
                  return a + b

Python feature since Python 3.7

VSCode understands type hints. It uses them for autocomplete and even warning for errors

Formatters rewrite code to make it follow a standard style. VSCode can run yours automatically.

Shift-Alt-F is the shortcut to run the formatter manually

Black is the best formatter

Linters look at your code and tell you problems including security issues. VSCode can run it automatically.

Quick fire VSCode features

The search feature looks through all files. You can exclude files, replace and use regex.

You can also use regex with Ctrl-F in files

Clicking while holding Alt allows you to create multiple cursors, and do super cool editing.

Pressing Shift-Alt-Right intelligently expands a selection

Pressing Ctrl-P quickly opens a file you recently had open

You can move lines by holding Alt and easily make copies holding Shift-Alt

In the Help menu there's a playground to try out some of these shortcuts

Some final extensions

Spell checker, pretty self explanatory.

Live share allows you to code along with someone else, facilitating pair programming

GitHub CoPilot uses AI to autocomplete code

Did I mention that VSCode was not built for Python?

VSCode is very popular for web development (JS, HTML, CSS) - you can also develop in R, C++. I use it with bash and tensorflow to build infrastructure.
